# thesis-reviewer

基于导师论文写作规范的硕士论文评审 Skill。

## 功能

对 LaTeX 硕士论文进行全面审查，按优先级输出问题：

| 级别 | 类型 | 说明 |
|------|------|------|
| **P0** | 致命问题 | 科学问题不清、逻辑断裂、抄袭风险 |
| **P1** | 重要问题 | 格式不一致、引用不规范、图表问题 |
| **P2** | 建议优化 | 表述优化、语言润色 |

## 评审维度

- 科学问题清晰度
- 章节逻辑连贯性
- 引用规范性
- 格式一致性
- 图表质量
- 语言表达

## 文件结构

```
thesis-reviewer/
├── SKILL.md                    # Skill 主文件（评审流程 + 报告格式）
└── references/
    └── checklist.md            # 60+ 详细检查项清单
```

## 使用方式

在 WorkBuddy 中加载 `thesis-reviewer` skill，然后对论文项目进行全面评审。

## 触发场景

- "帮我评审论文"
- "检查论文的逻辑问题"
- "审阅我的毕业论文"
- "论文格式检查"
- "论文修改建议"
