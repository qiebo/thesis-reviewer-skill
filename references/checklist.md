# 论文评审详细清单

## 一、科学问题与结构

### 1.1 科学问题
- [ ] 科学问题或技术问题是否清晰明确
- [ ] 是否是"属于你自己的小问题"而非公认的大问题
- [ ] 科学问题是否贯穿全文

### 1.2 章节结构
- [ ] 章节之间是否有承上启下的段落
- [ ] 每章开头是否总结前面章节、引出本章内容
- [ ] 每章结尾是否有过渡段落
- [ ] 全文是否围绕一个核心科学问题展开

### 1.3 逻辑连贯
- [ ] 段落之间过渡是否平滑
- [ ] 句子之间是否有上下文承接
- [ ] 是否有突兀的新概念引入（没有解释）
- [ ] "问题→挑战→内容→贡献"的链条是否清晰

## 二、引用与原创性

### 2.1 引用规范
- [ ] 别人的公式是否有引用
- [ ] 别人的图片是否有引用
- [ ] 别人的图表是否有引用
- [ ] 别人的表格格式是否有引用
- [ ] 所有借鉴的图片务必引用

### 2.2 原创性检查
- [ ] 是否有高度模仿他人图片的问题
- [ ] 是否有直接使用他人公式的情况（无引用）
- [ ] 是否有表格格式高度相似的情况

### 2.3 参考文献格式
- [ ] 期刊和期刊之间格式是否统一
- [ ] 会议和会议之间格式是否统一
- [ ] 年份、页码、卷号位置是否一致
- [ ] 会议简称是否统一（如NeurIPS）

## 三、写作格式

### 3.1 字体与符号
- [ ] 全文算法、模型、字体是否统一
- [ ] 公式字体是否统一
- [ ] 所有公式类字母是否添加$（如$xyz$）

### 3.2 标点符号
- [ ] 逗号使用是否合理（不超过2-3行）
- [ ] 句号使用是否充分
- [ ] 公式后标点是否统一
- [ ] 全角/半角是否统一
- [ ] 括号内英语大小写是否一致

### 3.3 编号与标题
- [ ] 小标题后句号是否统一（都要或都不要）
- [ ] 正文是否出现1、2、3等高层次编号
- [ ] (1)(2)(3)列表格式是否统一

### 3.4 图表格式
- [ ] 表格标题在上、图标题在下
- [ ] 先中文标题后英文标题
- [ ] 图表是否美观、占用空间合理
- [ ] 图中点和线粗细是否合适
- [ ] 表格是否占满页面空间

### 3.5 英文缩写
- [ ] 第一次出现的英文缩写是否有全称
- [ ] 简称只在正文第一次出现时写全称

## 四、语言质量

### 4.1 中文语言
- [ ] 是否有错别字
- [ ] 是否有口语化表达
- [ ] "其中"前面是否有空格
- [ ] 是否另起一段

### 4.2 英文语言
- [ ] Abstract是否经过语法检查
- [ ] 是否使用this thesis/study而非this paper
- [ ] 小数点后位数是否统一
- [ ] 是否避免使用分号（英语中没有）
- [ ] Related work后是否加s
- [ ] Let后面动词是否加s

### 4.3 表述规范
- [ ] 是否使用"our method"（应改为"the proposed XXXX"）
- [ ] 是否有"that is"等口语化表达
- [ ] 是否有"some may"等含糊词
- [ ] 是否有超过4行的长句子

## 五、格式细节

### 5.1 引用格式
- [ ] \cite{}前面空格是否统一
- [ ] 公式引用Eq.~eqref是否规范统一
- [ ] e.g.和i.e.用法是否统一（i.e.后必须有逗号）

### 5.2 用词统一
- [ ] module和block等同类词是否统一
- [ ] 斜体使用是否统一（如\emph{...}）

### 5.3 查重相关
- [ ] 重复超过0.01%的内容是否都有引用
- [ ] 是否有完全重复的句子
- [ ] 是否有类似长句子未引用

## 六、自查建议

完成初稿后建议按以下清单核查：

1. 检查所有章节、图表、公式编号及引用是否对应
2. 核对目录、页眉、页码是否正确
3. 检查参考文献格式全文是否一致
4. 通读全文，修正错别字及格式问题
5. 确保图表标题无误且无跨页问题
6. 检查图、表清单及变量注释表
7. 对比摘要、引言、文献综述确保内容不重复
8. 审视选题范围与论述深度
9. 评估文献综述是否做到"述、评、创"结合
10. 请他人协助检查，避免自查疏忽
